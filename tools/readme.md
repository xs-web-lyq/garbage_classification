
# 📖 README - 数据集预处理工具

## 🚀 项目概述
本工具用于自动化处理图像分类任务的数据集准备，主要功能包括：
1. **自动解析文件夹结构**：根据预设的类别标签生成标注文件
2. **数据集划分**：按指定比例分割训练集和验证集
3. **生成标准标签文件**：输出JSON格式的标注文件

## 📂 数据准备要求
1. **目录结构**：
   ```
   dataset/
   └── images/
       ├── apple/
       │   ├── img001.jpg
       │   └── ...
       ├── banana/
       ├── battery/
       └── battle/
   ```
